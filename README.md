# 뽀모도로 타이머

PC에서 브라우저로 바로 실행하는 뽀모도로 타이머입니다.
바이브 코딩으로 만들어보고 기능과 UI를 조금씩 다듬어 본 프로젝트입니다.

## 실행 방법

1. `start-pomodoro.cmd`를 더블클릭합니다.
2. 또는 `index.html`을 브라우저에서 엽니다.

## 기능

- 집중, 짧은 휴식, 긴 휴식 모드
- 시작, 일시정지, 리셋, 다음 모드 이동
- 집중/휴식 시간 설정 및 브라우저 저장
- 타이머 종료 시 알림음
- 자동 다음 모드 전환
- 데이 모드, 나이트 모드 색상 전환 및 브라우저 저장
- 애플 스타일의 블루 포인트 다크 모드
- 화면 오른쪽 위 데이/나이트 모드 버튼
- 타이머 숫자 근처 PIP 버튼
- 타이머 숫자만 별도 PIP 화면으로 표시
- PC에서는 타이머를 왼쪽, 버튼과 설정을 오른쪽에 두는 반응형 레이아웃
- 화면 폭이 좁아지면 세로형 모바일 레이아웃으로 자동 전환

## 파일 구조

- `index.html`: 화면 구조와 스크립트 연결
- `style.css`: 데이/나이트 색상, 반응형 레이아웃, 버튼 스타일
- `js/config.js`: 기본 시간, 모드 순서, 표시 문구 설정
- `js/storage.js`: 브라우저 저장소 로드/저장
- `js/theme.js`: 데이/나이트 모드 적용
- `js/audio.js`: 타이머 종료 알림음
- `js/pip.js`: 타이머 숫자 PIP 표시
- `js/app.js`: 타이머 상태, 버튼 이벤트, 화면 갱신
- `start-pomodoro.cmd`: 브라우저에서 앱을 여는 실행 파일

## 참고

- PIP 기능은 브라우저의 PIP 지원 여부와 팝업 차단 설정에 따라 동작 방식이 달라질 수 있습니다.
- Chrome/Edge 계열에서는 가능한 경우 영상 PIP처럼 타이머 숫자만 띄우도록 동작합니다.
